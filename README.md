# node-graphql-mysql

Dockerizable Node app with Mysql and GraphQL server
This repo is the end-point of rest api to "next-react-ts" repo

## Technologies Used

Node, Express
Mysql, GraphQL
Typescript
Jest
Docker
Socket

#### app

RUN

1. yarn install
2. npm run dev

Test

1.  npm run test

Docker

1. docker-compose up --build
