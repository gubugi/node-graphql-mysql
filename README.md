# node-graphql-mysql

Dockerizable Node app with Mysql and GraphQL server
This repo is the end-point of rest api to "next-react-ts" repo

### Technologies Used

1. Node, Express
2. Mysql, GraphQL
3. Typescript
4. Jest
5. Docker
6. Socket
